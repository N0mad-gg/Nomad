# Nomad

A lightweight, decentralized voice & chat platform for gamers.

> Discord's UX. Your data.

## Stack

- **Frontend**: React + Tauri
- **Backend**: Go + WebTransport
- **Database**: Redis + PostgreSQL
- **Blockchain**: Solana
- **Infra**: Fly.io

## Structure

```
apps/
  desktop/     # React + Tauri desktop app
server/        # Go backend
contracts/     # Solana smart contracts
docs/          # Documentation
```
